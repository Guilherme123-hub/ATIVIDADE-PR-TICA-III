# ATIVIDADE PRÁTICA III

**Aluno:** Guilherme Martins
**Período:** 1º Período
**Disciplina:** Desenvolvimento Web I
**Descrição:** Resolução de 5 atividades práticas sobre layouts utilizando CSS Flexbox.

## Estrutura do Projeto
- `atividade1/`: Layout com menu lateral fixo e cabeçalho.
- `atividade2/`: Seção Hero e blocos com quebra de linha.
- `atividade3/`: Dashboard com blocos de diferentes proporções.
- `atividade4/`: Layout com abas e colunas automáticas.
- `atividade5/`: Grade (grid) de 6 blocos (3 por linha).
